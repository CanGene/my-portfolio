Graphic Design

By Candace Genesis

Postage Stamps

Project Brief: Using an existing photograph of an animal, reptile, or insect, you are to use Illustrator to redraw the animal based on the main components (shapes) that make up the image. You will then create a postage stamp design that incorporates this vector illustration.

Tools Used: Adobe Illustrator

Impossible Environment

Project Brief: You will be using image editing tools and techniques in Photoshop to combine features of three different images to make one single, believable, full-color, composite image that communicates an emotion or feeling. The final image will present an “impossible environment”, in which the objects could only make sense together within an imaginative or fantasy scenario. It should look believable, as if all three elements originally existed in a single original photograph.

Tools Used: Adobe Photoshop
